VAR is the oldest variable declaration keyword. VAR declaration can be global scope or function or local scope. It can be re-declared and updated. It is hoisted to the top of their scope. LET is a ES6 variable declaration keyword. It is a block scope varibale. It can be updated but not re-declared. LET declarations are hoisted to the top. CONST also a ES6 variable declaration keyword. It's declaration are block scoped. It cannot be updated or re-declared.
