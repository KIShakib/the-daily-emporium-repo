VAR is the oldest variable declaration keyword. VAR declaration can be global scope or function or local scope. It can be re-declared and updated. It is hoisted to the top of their scope.
LET is a ES6 variable declaration keyword. It is a block scope varibale. It can be updated but not re-declared. LET declarations are hoisted to the top.
CONST also a ES6 variable declaration keyword. It's declaration are block scoped. It cannot be updated or re-declared.

Noraml function & Arrow function are mainly different by their syntax. Arrow function is an ES6 feature. Arrow function don't have their own bindings to this, arguments or super, and should not be used as methods. This function don't have access to the new.target keyword. Arrow Function Syntax const arArrowFunction = (parameter) => (parameter * 2);
Function keyword can be used to define a regular function. It can be a anonymous function. It is the oldest system to define a function. Normal function syntax function functionName (parameter){return parameter * 2;};

The Map() method receives a function as a parameter. Then it applies it on each element and returns an new array with the results of calling the provided function.This means it returns a new array that contains a result of each element of the array. It will always return the same number of items.
The forEach() method receive a function as an argument and executes it once for each array element. But instead of returning a new array like map, it returns undefined. This is the main difference between Map() & forEach().
The filter() method return the all element is an array that which fulfilled the conditions.
The find() method return the first element from which fulfilled the conditions.
